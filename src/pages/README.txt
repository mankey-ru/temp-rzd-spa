Здесь предполагается размещать компоненты, являющиеся страницами SPA
Например, route, pass, cabinet.
У каждой страницы есть главный скрипт и три vue-файла для разных представлений, которые его используют в качестве миксина.
В качестве миксина - потому что его не выйдет расшарить, если просто писать <script src="./myscript">
